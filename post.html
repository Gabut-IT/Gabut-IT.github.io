Pengenalan PHP

PHP Merupakan server-side scripting language
Mirip dengan sintaks C, Java dan Perl
Sejarah:
♦ Dikembangkan oleh Rasmus Lerdorf pada tahun 1994
♦ Sekarang dikelola oleh PHP Group (http://php.net/)
Cross Platform :
♦ (Free) untuk Windows, Linux, Mac dan Unix
♦ Dapat bekerja dengan berbagai web server (contoh: IIS, Apache) dan sistem basisdata (contoh: MySQL, MS SQL Server, Oracle)

untuk dapat menjalankan skrip PHP membutuhkan hal berikut

WebServer (Seperti: Apache, Nginx, IIS, dll)

kamu juga akan membutuhkan sebuah Database seperti MariaDB atau MySQL sebagai fasilitas penyimpanan data.

Setelah semua tersedia, maka PHP akan melakukan tugasnya yaitu melakukan eksekusi terhadap semua file dengan tipe .php yang telah disimpan dalam direktori tertentu.
